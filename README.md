`apt-cache search libsdl2`

`apt-cache search libsdl2-image`

`sudo apt install libsdl2-image-dev`

#### SDL Fonts Install

`apt-cache search libsdl2-ttf`

`sudo apt install libsdl2-ttf-dev`
